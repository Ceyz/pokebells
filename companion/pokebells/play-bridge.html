This page is a wallet bridge. Install the Nintondo wallet extension and reload. Once the extension injects window.nintondo on this page, the bridge relays every call from the embedded game — so the game keeps working even when loaded from a Nintondo content host where the extension isn't injected.

This file is a generic template: anyone can host it (Cloudflare Pages, GitHub Pages, a local python -m http.server, a community mirror, …) and point users at their URL via ?bridge=<their-url>. No server logic, no state — just a postMessage relay to the browser's wallet extension.
